# 游玩教程
## 1.下载Minecraft
下载[PCL2启动器](https://afdian.net/p/0164034c016c11ebafcb52540025c377)，安装1.20.4版本

## 2.进行正版/离线登陆
然后打开游戏
## 3.进入服务器
打开游戏后，点击多人游戏，添加服务器，输入服务器地址(如下)后，双击进入服务器

🔀 服务器有多条线路，会随机进行选择，如果您觉得这条线路有点卡，可以重进一下服务器
```plaintext
mc.mistmc.top
```
